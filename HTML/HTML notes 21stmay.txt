Attributes :
---------------------------
An HTML attribute is a piece of markup language used to adjust the behaviour display of any HTML element.
For example, attribute can be used to change the color, size, or functionality of the HTML element.

List items
-----------------------------
3 types
1)order list item
	<ol> </ol>

2) Unordered list item
	<ul> </ul>
3) Description lists
	<dl> </dl>     ----the <dl> tag defines the description list
	<dt> </dt>     ----the <dt> tag defines the term (name), and
	<dd> </dd>     ----the <dd> tag defines each term




  <body>
